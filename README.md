# Bouvet Battle Royale 2017 entry
`> npm install`
`> node index.js`
Serveren kjører nå på localhost:3002

Installer ngrok
`./ngrok http 3002 --host-header="localhost:3002"`

Du får nå en url type http://603b4ba1.ngrok.io som kan brukes i battle
